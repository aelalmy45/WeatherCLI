# WeatherCLI

A beautiful terminal weather dashboard built with Python and Rich.

  ![Screenshot from WeatherCLI](images/Screenshot.jpg)

## Features

- Current weather
- Feels like temperature
- Humidity
- Wind speed
- UV Index
- Sunrise & Sunset
- 24-hour day progress bar
- Rich terminal UI
- Works in Termux

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```python
python dashboard.py
```
